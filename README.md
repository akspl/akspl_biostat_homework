# Биостатистика и анализ медицинских данных 

Репозиторий содержит в себе практические задания, выполненные в рамках курса ‘Биостатистика и анализ медицинских данных’ (2025/2026) от [Института биоинформатики](bioinf.me/education/stat). 

Задания выполнялись в RStudio c использованием языка R. При выполнении всех заданий использовался пакет tidyverse.

**multifactorial_analysis** - в рамках задания создавали полноценный отчет, содержащий результаты оценки общего эффекта курения матери во время беременности на исходы родоразрешения. Оценку проводили с помощью линейной и логистической регрессии с коррекцией на отобранные с помощью DAG конфаундеры. В работе использовали пакеты lmtest, emmeans, sandwich и car.

**homework_classical_tests** - практика использования классических статистических тестов (Т-тест Стьюдента, тест Манна-Уитни, тест Уилкоксона, критериq Хи-квадрат, коэффицент корреляции Пирсона и Спирмена) и расчета RR и OR.

**quant_research_planning** – практика расчета объема выборки исследования с использованием пакетов TrialSize и epiR. Файл quant_rearch содержит код расчетов выборок для исследований превосходства и не меньшей эффективности.

**ADaM coding** – практика основ статистического программирования. Файл admh.Rmd содержит код сборки датасета с анамнезом субъектов согласно спецификации.

**specifics_of_med_data_task** - практика построения и анализа ROC-кривой с использованием пакета pROC.

**survival_analysis** – практика построения и интепретации кривой Каплана-Мейера и Hazart rate с применением пакетов survival, survminer и biostat3.

**basic_datavis_hw1_2025** и **basic_datavis_hw2_2025** – практика базовой визуализации данных с применением пакетов ggplot2, patchwork, ggsignif, gghighlight, treemapify и ggimage. Файлы BI_retarining_datavis_hw_1.qmd и BI_retarining_datavis_hw_2.qmd в соответствующих папках содержат в себе код для базовой визуализации данных.

**MultidimentionalAnalysis_HW** - практика применения методов многомерного анализа (PCA), кластеризации и классификации данных с использованием пакетов factoextra, pheatmap, GGally и umap.

**probability_theory** – практика создания виртуальный эксперимента для анализа возникновения ошибки в оценке вероятности события в зависимости от истинной вероятности и объема выборки. Файл Estimation_of_probability.rmd содержит код для проведения виртуального эксперимента.

В начале каждого файла с кодом содержится информация о датасете и расположении ключевых файлов при необходимости.
